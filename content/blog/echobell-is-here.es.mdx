# ¡Echobell está aquí!

Hoy, nos complace anunciar el lanzamiento oficial de Echobell – una revolucionaria aplicación de alertas instantáneas diseñada para transformar la forma en que recibes y gestionas las notificaciones.

## ¿Qué es Echobell?

Echobell es una aplicación de alertas instantáneas que aporta simplicidad y poder a la gestión de notificaciones. En el mundo actual de sobrecarga de información, mantenerse al día con las alertas verdaderamente importantes puede ser un desafío. Echobell resuelve esto proporcionando una plataforma flexible que puede manejar webhooks o correos electrónicos como disparadores, y luego enviar notificaciones personalizadas a ti y a tus suscriptores.

El principio fundamental detrás de Echobell es sencillo: crea canales para diferentes tipos de notificaciones, personaliza cómo aparecen esas notificaciones y elige con qué urgencia deseas ser alertado.

## Características Principales

### Canales de Notificación

En el corazón de Echobell están los **Canales** – conductos de notificación personalizables que puedes crear para diferentes propósitos. Cada canal tiene:

- Un nombre y color único para fácil identificación
- Plantillas de notificación personalizables que determinan qué información se envía
- Opciones flexibles de activación vía webhooks o correo electrónico
- Enlaces de suscripción compartibles para que otros puedan recibir tus notificaciones

### Múltiples Métodos de Activación

Echobell ofrece dos formas poderosas de activar notificaciones:

**Webhooks**: Integra con prácticamente cualquier sistema usando nuestras URLs de webhook. Cuando estas URLs son llamadas (con una simple solicitud GET o POST), tu canal envía notificaciones a todos los suscriptores. Esto es perfecto para:

- Alertas de monitoreo de servidores
- Notificaciones de pipeline CI/CD
- Actualizaciones de estado de dispositivos IoT
- Notificaciones de pedidos de comercio electrónico

**Activadores por Correo Electrónico**: Cada canal puede tener una dirección de correo electrónico dedicada. Cuando se envía un correo a esta dirección, el canal se activa y las notificaciones son enviadas. Esto facilita:

- Reenviar correos importantes como notificaciones
- Crear flujos de notificación sin programación
- Integrar con sistemas heredados que soportan correo pero no webhooks

### Plantillas de Notificación Personalizables

Haz que tus alertas sean significativas usando nuestro sistema de plantillas con variables dinámicas:

```
Título: Alerta de estado de {{server}}
Cuerpo: {{message}} detectado en {{timestamp}}
```

Cuando se activa con variables como `server=producción` y `message=Uso de CPU al 95%`, la notificación se vuelve personalizada e informativa.

### Tipos de Notificación Flexibles

No todas las alertas son iguales. Echobell permite a los suscriptores elegir entre tres tipos de notificación:

- **Normal**: Notificaciones estándar de la aplicación para alertas rutinarias
- **Sensible al Tiempo**: Notificaciones de alta prioridad que pueden atravesar ciertos modos de concentración
- **Llamada**: Alertas tipo llamada para situaciones críticas que requieren atención inmediata

### Enfoque Centrado en la Privacidad

Hemos construido Echobell con la privacidad como núcleo:

- Solo las cuentas de usuario, canales y datos de suscripción se almacenan en nuestros servidores
- El contenido e historial de notificaciones permanece exclusivamente en tu dispositivo
- Tus datos sensibles de alerta nunca salen de tu control

## Casos de Uso

Echobell es lo suficientemente versátil para servir a una amplia gama de necesidades de notificación:

### Para Desarrolladores

- Monitorea la salud del servidor y recibe alertas instantáneas cuando surgen problemas
- Recibe notificaciones cuando las compilaciones se completan o las pruebas fallan
- Recibe alertas cuando ocurren errores críticos en producción

### Para Traders

- Recibe alertas en tiempo real para cambios en precios de acciones
- Monitorea precios de criptomonedas y recibe notificaciones de fluctuaciones significativas
- Configura alertas para artículos de noticias o menciones en redes sociales de acciones específicas

### Para Equipos

- Crea canales para diferentes proyectos o sistemas
- Permite que los miembros del equipo se suscriban con su nivel de urgencia preferido
- Asegura que las alertas críticas nunca se pierdan, incluso durante tiempo de concentración

### Para Uso Personal

- Recibe alertas cuando tu sitio web se cae
- Recibe notificaciones de correos importantes sin revisar tu bandeja de entrada
- Crea alertas personalizadas para cualquier cosa importante para ti

## Primeros Pasos

Comenzar con Echobell es simple:

1. Descarga Echobell desde la [App Store](https://apps.apple.com/app/id6743597198)
2. Crea una cuenta usando correo electrónico o Iniciar Sesión con Apple
3. Crea tu primer canal con plantillas personalizadas
4. Configura un webhook o activador por correo electrónico
5. ¡Comienza a recibir notificaciones!
