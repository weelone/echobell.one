# 介绍 Echobell

Echobell 是一款注重隐私的 iOS 通知应用，通过 Webhook 或邮件接收触发信号，并以普通、时间敏感或来电式通知的形式推送到你的设备。你可以将提醒组织成频道，用模板定制内容，并为每个频道选择不同的紧急程度。

## 核心功能

### 频道

频道是 Echobell 的核心单元，每个频道包含：

- 名称和颜色，方便快速识别
- 带有变量的标题和正文模板（如 `{{server}}`、`{{message}}`、`{{timestamp}}`）
- 用于触发的 Webhook URL 或专用邮件地址
- 可分享的订阅链接，让团队成员订阅

### Webhook 和邮件触发

**Webhook：** 使用 GET 或 POST 请求调用频道的唯一 URL，Echobell 会通知所有订阅者。适合服务器监控、CI/CD 流水线、IoT 状态更新和订单通知等场景。

**邮件转通知：** 将邮件发送或转发至频道地址即可触发通知——无需编写代码。适合转发重要邮件、与旧系统集成，或在不写代码的情况下构建通知流程。

### 模板与变量

```
Title: {{server}} status alert
Body: {{message}} detected at {{timestamp}}
```

通过 Webhook 请求体或查询参数传入变量。对于邮件触发，`from`、`subject`、`text` 等变量会自动可用。

### 通知类型

- **普通（Active）：** 适用于日常更新的标准通知
- **时间敏感：** 可穿透专注模式
- **来电：** 像电话一样响铃，适用于紧急事件

### 隐私保护

账户、频道和订阅存储在服务端。通知内容和历史记录保留在你的设备上。

## 使用场景

**开发者：** 服务器和可用性监控（Grafana、Prometheus）、CI/CD 和 GitHub Actions 告警、生产错误上报。

**交易者：** 股票或加密货币价格阈值告警、通过邮件转发信号。

**团队：** 每个项目一个频道，团队成员按自己偏好的紧急级别订阅。

**个人：** 网站宕机告警、重要邮件通知、自定义提醒。

## 快速开始

1. 从 [App Store](https://apps.apple.com/app/apple-store/id6743597198?pt=128151925&ct=blog-echobell-is-here-zh&mt=8) 下载 Echobell
2. 创建账户（邮箱或 Apple 登录）
3. 创建频道并设置标题/正文模板
4. 添加 Webhook 或邮件触发
5. 发送测试通知
