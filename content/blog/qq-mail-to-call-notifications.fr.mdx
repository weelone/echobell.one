Les notifications push de QQ Mail sont faciles à manquer pendant un appel, en mode de travail concentré ou quand votre téléphone est en silencieux. Ce guide montre comment transférer uniquement les messages QQ Mail haute priorité vers Echobell pour déclencher une alerte en mode appel sur votre téléphone.

Documentation utile :

- [Configuration QQ Mail](/en/docs/email-to-call/qq)

## Prérequis

- Echobell installé ([App Store](https://apps.apple.com/app/apple-store/id6743597198?pt=128151925&ct=blog-qq-mail-to-call-notifications-fr&mt=8))
- Accès aux paramètres QQ Mail

## Étape 1 — Créer un canal pour le courrier urgent QQ

1. Ouvrez Echobell et créez un nouveau canal (ex. `QQ Critical Mail`).
2. Réglez le type de notification sur **Appel**.
3. Activez optionnellement les tentatives automatiques dans les paramètres de l'app pour une livraison plus robuste des messages critiques.

Voir : [Types de notifications](/en/docs/notification)

## Étape 2 — Copier l'email du canal et configurer les modèles

Depuis la page de détail du canal, copiez l'adresse email liée au canal. QQ Mail transférera vers cette adresse.

Gardez les modèles courts — les notifications d'appel lisent ce contenu à voix haute.

Modèle de titre :

```txt
[QQ Important Mail] {{subject}}
```

Modèle de corps :

```txt
From {{from}}. {{text}}
```

Référence des variables : [Modèles](/en/docs/template)

## Étape 3 — Créer des règles de transfert QQ Mail

Dans les paramètres QQ Mail, créez un filtre pour les messages haute priorité :

- Filtrer par expéditeur (ex. finance, sécurité, clients clés)
- Filtrer par mots-clés dans l'objet (`urgent`, `invoice`, `failure`, `verification`)
- Action : Transférer vers l'adresse email de votre canal Echobell

**Première vérification :** quand vous activez le transfert, QQ Mail envoie un email de vérification. Ouvrez cette notification dans Echobell, appuyez sur le lien en haut à droite pour afficher le corps complet de l'email, puis cliquez sur le lien de vérification pour activer le transfert.

Commencez avec une règle stricte. Ajoutez-en d'autres après avoir confirmé que la première fonctionne sans bruit excessif.

Guide complet spécifique à QQ : [/docs/email-to-call/qq](/en/docs/email-to-call/qq)

## Étape 4 — Ajouter des Conditions Echobell (optionnel)

Utilisez les Conditions comme filtre secondaire quand les règles QQ Mail ne peuvent pas exprimer la logique dont vous avez besoin.

Notifier uniquement pour un expéditeur spécifique :

```js
from == "billing@company.com"
```

Notifier uniquement quand l'objet contient « urgent » :

```js
subject.toLowerCase().includes("urgent")
```

Expéditeur et mot-clé combinés :

```js
(from == "security@company.com" || from == "ops@company.com") && subject.toLowerCase().includes("alert")
```

Référence : [Conditions](/en/docs/conditions)

## Note sur la confidentialité

Filtrez d'abord dans QQ Mail afin que seuls les emails nécessaires parviennent à Echobell. Si vous déployez ce workflow pour une équipe, informez les utilisateurs que le contenu de certains emails est transféré à des fins de livraison des notifications, conformément à votre politique interne et aux exigences locales.

## Étape 5 — Tester de bout en bout

1. Envoyez un email de test dans QQ Mail qui correspond à votre règle de transfert.
2. Confirmez que QQ transfère bien vers l'email de votre canal.
3. Confirmez qu'Echobell déclenche une notification d'appel.
4. Ajustez les modèles et les filtres en fonction de ce que lit l'appel.

## Résolution des problèmes

- **Pas d'appel reçu :** Vérifiez que le type d'abonnement est réglé sur Appel ; vérifiez les autorisations de notification iOS et le comportement du mode Concentration.
- **Trop d'appels :** Resserrez d'abord les règles QQ Mail. Ajoutez les Conditions Echobell uniquement si ce n'est toujours pas suffisant.
- **Contenu vocal trop long :** Raccourcissez le modèle et ne gardez que les champs essentiels.

## Prochaines étapes

- Créez des canaux séparés par thème (finance, sécurité, clients)
- Utilisez le push pour les mises à jour courantes, réservez les appels aux messages vraiment urgents
- Révisez les règles de transfert chaque mois
