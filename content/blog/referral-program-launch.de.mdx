Das Echobell-Empfehlungsprogramm ist gestartet. Teilen Sie Ihren Einladungscode, sammeln Sie Punkte und lösen Sie diese gegen kostenloses Premium ein.

### So funktionieren die Punkte

- **Sie verdienen** 20 Punkte, wenn ein Freund einen [Monatsplan](/en#pricing) abonniert oder verlängert, und 200 Punkte bei einem [Jahresplan](/en#pricing).
- **Ihr Freund verdient** 100 Punkte bei Verwendung Ihres Einladungscodes innerhalb der ersten 30 Tage.

### Wo Sie Ihren Code finden

Gehen Sie in der [Echobell-App](https://apps.apple.com/app/id6743597198) zu **Einstellungen → Freunde einladen**, um Ihren Code zu generieren oder den Code einer anderen Person einzugeben.

### Punkte einlösen

- **200 Punkte** = 1 Monat Premium
- **2000 Punkte** = 1 Jahr Premium

### Voraussetzungen

Vor dem Start auf Echobell **Version 1.22** oder höher aktualisieren.

### Bereits Nutzer?

Bestehende Nutzer können bis zum **15. Januar 2026** einen Einladungscode eingeben, um den 100-Punkte-Willkommensbonus zu erhalten.
