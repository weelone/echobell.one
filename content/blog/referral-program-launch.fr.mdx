Le programme de parrainage Echobell est lancé. Partagez votre code d'invitation, gagnez des points et échangez-les contre du Premium gratuit.

### Comment fonctionnent les points

- **Vous gagnez** 20 points à chaque fois qu'un ami souscrit ou renouvelle un [Forfait mensuel](/en#pricing), et 200 points pour un [Forfait annuel](/en#pricing).
- **Votre ami gagne** 100 points en utilisant votre code d'invitation dans ses **30 premiers jours**.

### Où trouver votre code

Allez dans **Paramètres → Inviter des amis** dans l'[application Echobell](https://apps.apple.com/app/apple-store/id6743597198?pt=128151925&ct=blog-referral-program-launch-fr&mt=8) pour générer votre code ou saisir celui d'un autre utilisateur.

### Échangez vos points

- **200 points** = 1 mois de Premium
- **2 000 points** = 1 an de Premium

### Conditions requises

Mettez à jour Echobell vers la **version 1.22** ou supérieure avant de commencer.

### Déjà utilisateur ?

Les utilisateurs existants peuvent saisir un code d'invitation avant le **15 janvier 2026** pour réclamer le bonus de bienvenue de 100 points.
