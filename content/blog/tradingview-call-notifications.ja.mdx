# もう取引の機会を逃さない：TradingView のアラートを Echobell で緊急電話通知として受信！

画面から目を離していたために重要な市場の動きを見逃してしまった経験はありませんか？重要な TradingView のアラートがトリガーされた瞬間に*実際の電話*を受けることができたらと想像してみてください！Echobell なら、それは夢ではなく現実です。見逃しやすい通知に頼るのをやめ、注目を要求するアラートを受け取り始めましょう。

以下は、TradingView のシグナルと Echobell の強力な通話アラートを連携させる方法です：

## 準備：必要なもの

- スマートフォンに Echobell アプリをインストールしたアカウント
- TradingView アカウント（注意：TradingView の Webhook 機能は通常、Pro、Pro+、または Premium サブスクリプションが必要です）

## ステップ 1：Echobell アラートチャンネルの作成

1. **新しいチャンネルの作成：** Echobell アプリで「チャンネル作成」をタップし、「TradingView GOLD アラート」などの印象的な名前を付け、アクションを示す色を選択します
2. **「通話」機能の有効化：** これが重要です！チャンネルを設定（または購読）する際、通知タイプとして**「通話」**を選択します。控えめな通知音ではなく、完全な通話アラートが必要です
3. **アラートメッセージの作成（忍者テクニック）：** 通話通知で*正確に*何が起きているのかを伝えたいですか？`タイトル`と`本文`のテンプレートをカスタマイズしましょう。TradingView はティッカーシンボルや価格などのデータを送信できます。電話が鳴り、Echobell が「アラート：BTC/USD が$50,000 を突破！」と告知する様子を想像してください。使用可能なアラート変数（`{{ticker}}`、`{{close}}`など）については TradingView のドキュメントを確認し、Echobell のテンプレートで`{{変数名}}`として使用します
4. **秘密の武器を入手：Webhook の URL：** チャンネル設定が完了したら、その詳細に移動します。**Webhook**のトリガー URL を見つけてください - このユニークなリンクがあなたの秘密の鍵です。慎重にコピーして大切に保管してください！TradingView はこの正確な URL を使用して Echobell の通話を即座にトリガーします

## ステップ 2：TradingView の設定

1. **戦場に参入：TradingView：** TradingView を開き、選択したチャートに移動します
2. **罠を仕掛ける：アラートの作成/編集：**
   - 時計アイコンの「アラート」をクリックするか、チャート、インジケーター、または描画物を右クリックしてアラート設定を呼び出します
3. **トリガーを定義：** レジスタンスレベルを突破する価格、お気に入りのインジケーターからのシグナルなど、「アラート！」と叫ぶべき市場条件を正確に設定します
4. **配線を接続：通知の設定：**
   - 「通知」または「アラートアクション」セクションを探します
   - **「Webhook URL」**のチェックボックスを見つけてチェックを入れます。これで接続が有効になります
   - ステップ 1 で取得した Echobell の**Webhook トリガー URL**をこのフィールドに直接貼り付けます
5. **メッセージの強化（オプションだが素晴らしい）：**
   - 「メッセージ」ボックスは、TradingView が\_何を\_Echobell に叫ぶかを指示する場所です
   - Echobell のテンプレートを輝かせるために、このメッセージを JSON 形式でフォーマットします。これによりチャートから動的なデータを直接電話通知に渡すことができます！
   - 典型的な JSON 構造は次のようになります（TradingView 固有のプレースホルダーを確認してください）：
     ```json
     {
       "ticker": "{{ticker}}",
       "price": "{{close}}",
       "action": "{{strategy.order.action}}"
     }
     ```
   - シンプルなメッセージでも問題ありませんが、JSON を使用することで、Echobell のテンプレートと組み合わせることで真に情報豊富な通話アラートが実現できます
6. **アラートを武装：** 「作成」または「保存」を押すと、TradingView のアラートが Echobell を通じて通話する準備が整います！

## ステップ 3：真実の瞬間 - テスト！

アラート条件が市場でトリガーされるのを待ちます（またはテストアラートを使用します）。準備してください！Echobell の独特な通話通知が、ほぼ瞬時にあなたのスマートフォンで鳴るはずです。（スマートフォンで Echobell に通知権限を付与することを忘れないでください！）

**おめでとうございます！** TradingView のパッシブな通知を、注目を集める無視できない電話通知に変換することに成功しました。よりスマートに取引し、より素早く反応し、二度と重要な市場イベントを見逃すことはありません！
