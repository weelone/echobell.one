[Uptime Kuma](https://github.com/louislam/uptime-kuma) 是一个易于使用的自托管监控工具。当您的服务宕机时，您可以通过 Echobell 接收通知。

<Steps>
  <Step>
    ### 创建频道
    在 Echobell 中创建一个新频道并复制 **Webhook URL**。
  </Step>
  <Step>
    ### 在 Uptime Kuma 中设置通知
    1. 转到 **设置** > **通知**。
    2. 点击 **设置通知**。
    3. 选择 **Webhook** 作为通知类型。
    4. 将 **显示名称** 设置为 "Echobell"。
    5. 将您的 Echobell Webhook URL 粘贴到 **Post URL** 中。
    6. 将 **请求方法** 设置为 `POST`。
    7. 将 **正文类型** 设置为 `JSON`。
    8. 启用 **自定义正文**。
  </Step>
  <Step>
    ### 配置 Payload
    在 **自定义正文** 字段中输入以下 JSON：

    ```json
    {
      "title": "Monitor {{name}} is {{status}}",
      "body": "{{msg}}\nURL: {{hostnameOrURL}}",
      "status": "{{status}}",
      "monitor": "{{name}}"
    }
    ```

    您可以根据需要自定义 JSON 结构。Echobell 将在您的通知模板中使用这些变量。

  </Step>
  <Step>
    ### 测试
    点击 **测试** 向 Echobell 发送测试通知。
  </Step>
</Steps>
