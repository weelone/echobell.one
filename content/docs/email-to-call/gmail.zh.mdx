# 将 Gmail 邮件转换为电话呼叫

按照本指南在 Gmail 中设置过滤器，自动将特定邮件转发到 Echobell，从而触发向您的设备拨打电话。

## 先决条件

- 已启用 **邮件触发** 的 Echobell 频道。
- 您的频道的唯一电子邮件地址（在频道设置中找到）。

## 第 1 步：添加转发地址

1.  在电脑上打开 **Gmail**。
2.  点击 **设置齿轮图标** (⚙️) 并选择 **查看所有设置**。
3.  转到 **转发和 POP/IMAP** 标签页。
4.  点击 **添加转发地址**。
5.  输入您的 Echobell 频道电子邮件地址（例如 `ch_... @trigger.echobell.one`）并点击 **下一步**。
6.  **验证：** Gmail 将向此地址发送验证码。
    - 打开 Echobell 应用。
    - 转到您频道的 **记录** 标签页。
    - 找到来自 Gmail 团队的邮件。
    - 点击邮件正文中的链接或复制确认码。
7.  返回 Gmail 设置，输入代码，然后点击 **验证**。

## 第 2 步：创建过滤器

请 **不要** 在常规设置中启用“将收到的邮件副本转发给”，因为这将转发 **所有** 邮件。请使用过滤器：

1.  点击 Gmail 搜索栏中的 **显示搜索选项** 图标（过滤器图标）。
2.  定义要触发电话的邮件标准（例如，发件人：`alerts@example.com`，主题：`Critical`）。
3.  点击 **创建过滤器**。
4.  勾选 **转发给：** 并选择您的 Echobell 频道地址。
5.  （可选）如果您不希望这些邮件出现在收件箱中，也可以勾选“标记为已读”或“存档”。
6.  点击 **创建过滤器**。

## 第 3 步：测试

发送一封符合过滤器标准的邮件到您的 Gmail 地址。您的手机应该会通过 Echobell 立即响起。

<Callout type="info">
  确保此频道的 Echobell 通知设置已设置为 **电话 (Calling)** 模式，以接收实际的电话呼叫。
</Callout>
