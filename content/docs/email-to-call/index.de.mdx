# E-Mail zu Anruf

Verwandeln Sie kritische E-Mails sofort in Telefonanrufe. Durch das Einrichten von Regeln zur automatischen Weiterleitung bei Ihrem E-Mail-Anbieter können Sie Echobell veranlassen, Ihr Telefon anzurufen, wann immer Sie eine dringende E-Mail erhalten.

## Warum E-Mail in Anruf umwandeln?

- **Keine Warnungen verpassen:** E-Mails können leicht übersehen oder ignoriert werden. Ein Telefonanruf ist schwer zu ignorieren.
- **Dringende Benachrichtigungen:** Perfekt für Serverausfallwarnungen, Sicherheitswarnungen oder kritische Geschäftsaktualisierungen.
- **Kein Programmieren erforderlich:** Verwenden Sie die integrierten Weiterleitungsregeln Ihres E-Mail-Anbieters.

## Wie es funktioniert

1.  **Holen Sie sich Ihre Kanal-E-Mail:** Kopieren Sie in Ihren Echobell-Kanaleinstellungen die eindeutige **E-Mail-Trigger**-Adresse (z. B. `ch_... @trigger.echobell.one`).
2.  **Weiterleitung einrichten:** Konfigurieren Sie Ihren E-Mail-Anbieter (Gmail, Yahoo, QQ usw.), um bestimmte E-Mails automatisch an Ihre Echobell-Kanaladresse weiterzuleiten.
3.  **Anrufe empfangen:** Wenn eine passende E-Mail eintrifft, wird sie an Echobell weitergeleitet, das sofort Ihr Telefon anruft.
