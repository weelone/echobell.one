# Convertir les emails Yahoo en appels téléphoniques

Apprenez comment utiliser les filtres Yahoo Mail pour transférer automatiquement les emails importants vers Echobell, les transformant en appels téléphoniques instantanés.

## Prérequis

- Un canal Echobell avec **Déclencheur Email** activé.
- Votre adresse email de canal Echobell.

## Étape 1 : Vérifier l'adresse de transfert

1.  Connectez-vous à **Yahoo Mail** sur le web.
2.  Allez dans **Paramètres** > **Autres paramètres**.
3.  Sélectionnez **Boîtes aux lettres**.
4.  Sous "Transfert", vous devrez peut-être ajouter et vérifier votre adresse Echobell.
    - *Remarque : Yahoo restreint parfois le transfert aux comptes payants. Si le transfert direct n'est pas disponible, vérifiez si les Filtres le permettent (voir ci-dessous).*
5.  Si une vérification est requise, vérifiez vos **Enregistrements** (Records) Echobell pour l'email de vérification de Yahoo.

## Étape 2 : Créer un filtre

1.  Allez dans **Paramètres** > **Autres paramètres**.
2.  Sélectionnez **Filtres**.
3.  Cliquez sur **Ajouter de nouveaux filtres**.
4.  Nommez votre filtre (par ex., "Alertes critiques").
5.  Définissez les règles pour les emails que vous souhaitez capturer (par ex., De contient `admin@monserveur.com`).
6.  Sous **Choisir un dossier de destination ou transférer**, sélectionnez **Transférer à** (si disponible).
7.  Entrez votre adresse email de canal Echobell.
8.  Cliquez sur **Enregistrer**.

<Callout type="warn">
  Les fonctionnalités de Yahoo Mail varient selon la région et le forfait. Si "Transférer à" n'est pas disponible dans les Filtres, vous pourriez avoir besoin de Yahoo Mail Plus.
</Callout>

## Étape 3 : Tester

Envoyez un email correspondant à votre adresse Yahoo et vérifiez que votre téléphone sonne.
