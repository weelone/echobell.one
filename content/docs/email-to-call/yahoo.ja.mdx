# Yahooメールを電話着信に変換する

Yahooメールのフィルタを使用して重要なメールを自動的にEchobellに転送し、即座に電話着信に変換する方法を学びます。

## 前提条件

- **メールトリガー** が有効になっているEchobellチャンネル。
- あなたのEchobellチャンネルのメールアドレス。

## ステップ 1：転送アドレスを確認する

1.  ウェブで **Yahooメール** にログインします。
2.  **設定** > **その他の設定** に移動します。
3.  **メールボックス** を選択します。
4.  「転送」の下で、Echobellアドレスを追加して確認する必要がある場合があります。
    - *注：Yahooは、有料アカウントにのみ転送を制限する場合があります。直接転送が利用できない場合は、フィルタで許可されているかどうかを確認してください（以下を参照）。*
5.  確認が必要な場合は、Yahooからの確認メールがないかEchobellの **記録** (Records) を確認してください。

## ステップ 2：フィルタを作成する

1.  **設定** > **その他の設定** に移動します。
2.  **フィルタ** を選択します。
3.  **新しいフィルタを追加** をクリックします。
4.  フィルタに名前を付けます（例：「緊急アラート」）。
5.  キャプチャしたいメールのルールを設定します（例：送信者に `admin@myserver.com` を含む）。
6.  **移動先フォルダを選択または転送** の下で、**転送先** を選択します（利用可能な場合）。
7.  Echobellチャンネルのメールアドレスを入力します。
8.  **保存** をクリックします。

<Callout type="warn">
  Yahooメールの機能は地域やプランによって異なります。フィルタで「転送先」が利用できない場合は、Yahoo Mail Plusが必要になる場合があります。
</Callout>

## ステップ 3：テストする

一致するメールをYahooアドレスに送信し、電話が鳴ることを確認します。
