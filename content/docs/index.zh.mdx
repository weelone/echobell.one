# Echobell 入门指南

欢迎使用 Echobell，这是一款帮助您及时处理重要通知的即时提醒应用。本指南将帮助您了解如何设置和有效使用 Echobell。

## 什么是 Echobell？

Echobell 是一款功能强大的通知管理工具，具有以下特点：
- 支持 webhook 和电子邮件作为触发器
- 向订阅者发送自定义通知
- 允许创建和分享通知频道
- 将您的通知数据私密保存在设备上

![Echobell 概览](/images/docs/screenshot-overview.en.png)

## 核心概念

在深入了解之前，让我们先理解一些关键概念：

### 频道

**频道**是 Echobell 中组织和分发通知的核心单位：
- 每个频道都有独特的**名称**和**颜色**，便于识别
- 频道包含决定发送信息内容的**通知模板**
- 频道可通过 webhook 或电子邮件触发
- 您可以创建自己的频道或订阅他人的频道

### 触发器

**触发器**是激活频道发送通知的方式：
- **Webhook**：当被调用时会激活频道的 URL 端点
- **电子邮件**：绑定到频道的电子邮件地址，收到邮件时会激活频道

### 通知类型

订阅频道时，您可以选择不同的通知样式：
- **普通**：标准应用通知
- **时效性**：可以突破专注模式的高优先级通知
- **来电**：用于最紧急事项的类电话提醒

## 开始使用

### 1. 设置账户

1. 从 [App Store](https://apps.apple.com/app/id6743597198) 下载 Echobell
2. 打开应用并使用您的电子邮件或 Apple 账号登录

![账户设置](/images/docs/screenshot-sign-in.en.png)

### 2. 创建您的第一个频道

1. 在主视图中，点击右下角的按钮进入**频道**视图
2. 点击**+**按钮创建新频道
2. 输入频道名称（例如"服务器告警"）
3. 选择一个颜色来标识您的频道
4. 配置您的通知模板：
   - **标题模板**：显示为通知标题
   - **内容模板**：通知的详细内容
7. 可选择是否订阅自己的频道
8. 为您的频道选择**通知类型**（普通、时效性或来电）
9. 保存新频道

![创建频道](/images/docs/screenshot-create-channel.en.png)

### 3. 设置触发器

创建频道后，有两种触发通知的方式：通过 webhook 或电子邮件。点击您刚创建的频道访问其设置。在频道设置中，您可以找到**触发器**部分。

![频道设置](/images/docs/screenshot-channel-settings.en.png)

#### Webhook 触发器

1. 点击 **webhook** 按钮复制生成的 webhook URL
2. 将此 URL 集成到您的系统中（如服务器监控、CI/CD 流程）
3. 当 URL 被调用时，您的频道将被触发

您可以通过两种方式为通知模板提供变量：

```bash
# 使用查询参数的 webhook 调用示例
curl -X GET "https://hook.echobell.one/t/your-unique-id?server=production&status=critical&message=CPU%20usage%20at%2095%25"

# 或使用 JSON 主体
curl -X POST https://hook.echobell.one/t/your-unique-id \
  -H "Content-Type: application/json" \
  -d '{"server": "production", "status": "critical", "message": "CPU usage at 95%"}'
```

这些变量将用于填充您的通知模板。例如，如果您的标题模板是 `{{server}} - {{status}}`，它将显示为 "production - critical"。

#### 电子邮件触发器

1. 点击**电子邮件**按钮复制为您的频道生成的电子邮件地址
2. 当有邮件发送到此地址时，您的频道将被触发

对于电子邮件触发器，Echobell 自动从邮件中提取以下变量：
- `from`：发件人的电子邮件地址
- `to`：收件人的电子邮件地址（您频道的邮件地址）
- `subject`：邮件主题行
- `text`：邮件的纯文本内容
- `html`：邮件的 HTML 内容

您可以在模板中使用这些变量，如 `{{subject}}` 或 `{{text}}`。

### 4. 自定义通知模板

模板使用从触发器数据填充的变量：

1. 在频道设置中，转到**通知模板**
2. 标题模板可以使用：`{{server}} - {{status}}`
3. 内容模板可以使用：`{{message}} 检测时间：{{timestamp}}`
4. 变量使用双大括号括起：`{{variable_name}}`
5. 使用示例数据测试您的模板

### 5. 订阅频道

1. 您可以与他人分享您的频道订阅链接
2. 或者使用他人分享的订阅链接来订阅频道
3. 在频道视图中点击**订阅**
4. 输入订阅链接
4. 选择您喜好的**通知类型**：
   - 普通用于标准提醒
   - 时效性用于重要提醒
   - 来电用于紧急情况
5. 确认您的订阅

## 隐私和数据存储

Echobell 优先考虑您的隐私：
- 只有用户账户、频道和订阅数据存储在我们的服务器上
- 通知内容和历史记录本地存储在您的设备上
- 您的提醒数据始终在您的控制之下
