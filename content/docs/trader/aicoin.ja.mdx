# AiCoin 連携

[AiCoin](https://www.aicoin.com) は、プロフェッショナルな暗号資産市場データおよび取引ツールです。Echobell と統合することで、AiCoin の価格警告や指標シグナルを即時の電話や通知として受け取ることができ、取引の機会を逃すことがありません。

<Steps>
  <Step>
    ## Echobell でチャンネルを作成する

    1. Echobell アプリを開きます。
    2. **+** ボタンをタップして新しいチャンネルを作成します。
    3. 「AiCoin アラート」などの名前を付けます。
    4. 好きなアイコンと色を選択します。
    5. チャンネルの詳細で、**Webhook URL** をコピーします。

  </Step>

  <Step>
    ## AiCoin アラートの設定

    1. AiCoin PC クライアントを開きます。
    2. 監視したい取引ペアの K ラインチャートに移動します。
    3. **アラート**（またはベルのアイコン）をクリックし、**アラートを追加**を選択します。
    4. トリガー条件を設定します（例：価格 > 60000）。
    5. 通知設定で、**Webhook** オプションにチェックを入れます。
    6. Echobell の Webhook URL をフィールドに貼り付けます。

  </Step>

  <Step>
    ## 統合のテスト

    アラートを保存します。条件が満たされると、AiCoin は Echobell にリクエストを送信し、デバイスで即座に通知または電話を受け取ります。

  </Step>
</Steps>
